# Prozorro Catalog — документація

API каталогу товарів і послуг для системи публічних закупівель Prozorro.

## Розділи

| Розділ | Опис |
|--------|------|
| [Краудсорсинг](crowdsourcing/README.md) | Реєстрація постачальників, подача та модерація заявок на товари |
| [Локалізація v3.0](vendor_to_crowdsourcing/README.md) | Актуальний флоу реєстрації виробників і локалізованих товарів через краудсорсинг |
| [Локалізація v3.0 — план змін](vendor_to_crowdsourcing/plan.md) | Технічний план міграції з v2.0 на v3.0 |
| [Локалізація v2.0](vendor_products/README.md) | ⚠️ Deprecated. Старий флоу через `/api/vendors` |
| [Теги](tags/README.md) | Управління тегами категорій і профілів |

## Швидкий старт

API доступне за адресою `https://market-api.prozorro.gov.ua/api`.

Інтерактивна документація (OpenAPI): `/api/doc`
